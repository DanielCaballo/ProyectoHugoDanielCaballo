<h1>Comment se déroule le processus de chargement ?</h1>

<h3>Eh bien, comme ceci : </h3>

1. La première chose est que le client (vous) Entrez le nom de la page Web.
2. Ensuite, **Votre** ordinateur envoie la demande à votre routeur.
3. Votre routeur envoie une requête au serveur DNS :
    + Le nom que vous mettez n'est pas celui dont vous avez besoin.
    + Il a besoin de l'adresse IP qui est échangée par DNS.

4. Le DNS Trouvez le serveur auquel vous souhaitez vous connecter.
5. Renvoyez les informations à votre routeur.
6. Votre routeur envoie les informations à votre ordinateur.
7. Et c'est prêt pour que vous puissiez voir les informations du serveur.
